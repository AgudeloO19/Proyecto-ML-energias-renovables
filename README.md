# 🔋 Renewable Energy Consumption Prediction (ML Project)

## 📌 Descripción

Este proyecto desarrolla un modelo de Machine Learning para predecir el consumo de energías renovables en Estados Unidos, utilizando datos históricos de la U.S. Energy Information Administration (EIA).

Se sigue la metodología **CRISP-DM**, abarcando desde el análisis exploratorio hasta la evaluación de modelos predictivos.

---

## 🎯 Objetivo

Predecir el consumo total de energías renovables (BTU) para apoyar la toma de decisiones en planificación energética y sostenibilidad.

---

## 🧠 Modelos utilizados

* Regresión Lineal
* RidgeCV (modelo principal)
* Random Forest
* XGBoost

---

## 📊 Técnicas aplicadas

* Análisis exploratorio de datos (EDA)
* Limpieza y preprocesamiento
* Ingeniería de características
* Normalización de datos
* One-Hot Encoding
* Evaluación con métricas (R², RMSE, MAE)

---

## 📁 Estructura del proyecto

```

reports/ → resultados y visualizaciones

```

## 📈 Resultados

El modelo RidgeCV mostró el mejor balance entre precisión e interpretabilidad, alcanzando un alto nivel de ajuste en la predicción del consumo energético.
